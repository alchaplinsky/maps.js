Easier Way to Work with Google Maps API

Maps.js is a Javascript library that gives you easy and declarative way to work with Google Maps Javascript API V3. Creating maps and adding markers is just as easy as passing options object literal to a constructor function.

Download v0.1.0 View on GitHub


Getting Started

1. Simply add Google Maps script to your page:

<script src="http://maps.google.com/maps/api/js?sensor=true"></script>

2. Add a container for map with fixed height to your HTML:

<div id="worldmap" style="height: 500px;"></div>

3. Create a new instance of Map object with parameters:

map = new Map({ div: '#worldmap', lat: 20, lng: 20 });

Have Fun with Google Maps!
